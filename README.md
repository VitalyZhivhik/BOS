Мне необходимо написать программу. Она состоит из двух частей.
Первая установлена на сервере она анализирует ПО которое установлено на сервере, средства обеспечения безопасности которые есть, какая есть инфраструктура на сервере, с помощью чего она защищается.
Втораячасть уставлена на компьютере якобы атакуюешго.
Она проверяетет порты делает тому подобные действия, вообщем ищет способы атаки на сервер. Обе части программы соеденены между собой.
Программы которая устанволена на компьютере атакующей команды отправляет список атак на сервер.
На сервере есть база данных с CVE (Common Vulnerabilities and Exposures) (https://cve.org),
база CWE (Common Weakness Enumeration) (https://cwe.org),
база  CAPEC (Common Attack Pattern Enumeration and Classification)
и база Матрицы MITRE ATT&CK (Adversarial Tactics, Techniques, and Common Knowledge) (https://attack.mitre.org).
Программа производит сопоставлние этих 4 баз данных с возможными способами атаки на сервер например резльтат работы ScanOval и
анализом ПО которое установлено на сервере, средств обеспечения безопасности которые есть, какая есть инфраструктура на сервере, с помощью чего она защищается.
Далее происхоит отчёт о том какие атаки не реализуемы в нашей системы. Например ScanOval нашёл уязвимость что наш сервер открыт для SQL инекций,
однако на нашем сервере нет ни одной базы данных поэтому данныя атака не имеет смысла. Также необходим отчёт о том какие действия надо совершить на сервере чтобы защититься от компьютерных атак.
